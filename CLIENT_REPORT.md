## Risk Analysis

- Generated by LLM after calculations complete
- **6 risk factors** per deal, each with:
  - Category badge (Financial / Operational / Market / Legal)
  - Source badge: **Memo** (document-grounded) or **General** (industry knowledge)
  - Confidence score (0–100%)
  - Citation from CIM text
- Displayed below Validation Summary on the Analysis page
- Based on actual calculated outputs: MOIC, FCCR, DSCR, Adj. EBITDA, Total Debt Service

## Key Deal Metrics (Calculated per Run)

| Metric | Description |
|---|---|
| MOIC | Multiple on Invested Capital |
| FCCR | Fixed Charge Coverage Ratio — pass if ≥ 1.0 |
| DSCR | Debt Service Coverage (Y1–Y5) |
| Total Sources | Sum of all ABL + debt financing lines |
| Total Uses | Purchase price + fees + working capital |
| Balance Check | Sources = Uses verified automatically |
